Paginas Web


Caracteristicas





En Internet encontramos muchas características de una página web, pero nos enfocaremos
en 3 características básicas:

Conectividad: En la actualidad los hipervínculos hacen que un documento sea dinámico,
el usuario puede ejecutar diferentes acciones a través de textos, imágenes, videos, gifs,
entre otros una y otra vez, haciendo posible la visualización de información de uno y otro
autor, iniciar un correo electrónico, un mensaje, una video conferencia, los hipervínculos
hacen una característica peculiar de las páginas web.

Accesibilidad: cualquier página web tiene información compilada que posteriormente se
presenta mediante un ordenador que está conectado a internet, que permite que se vea.

Interactividad: En la actualidad no hay un límite puesto a las actividades que se puedan hacer
en la web, el usuario puede visualizar información a color, con animaciones, imágenes e interactuar
entre páginas, realizar compras, consultas, comentarios o hasta enviar archivos en línea a nivel mundial.
